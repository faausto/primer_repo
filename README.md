# **Práctico Integrador — Git y GitHub**
## Descripción
Repositorio de práctica para las clases 1, 2 y 3 de Software Factory.
Contiene ejercicios de Git, GitHub y Python básico.
## Contenido
- `mi_info.py` — Script con información personal y hobbies
- `calculadora.py` — *Aplicacion basica de calculadora*
- `quiz_git.py` — Quiz interactivo sobre conceptos de Git
- `.gitignore` — *Archivos ignorados por Git*

## Cómo ejecutar

**Requisito: tener Python 3 instalado.**

Todos los archivos `.py` de este repositorio son independientes, por lo que se ejecutan de la misma manera desde la terminal de cada sistema operativo.

**En la terminal de Windows:**
`python nombre_de_archivo.py`
o también puede ser:
`py nombre_de_archivo.py`

*Ejemplos:* 
*python calculadora.py*
*py mi_info.py*

**En la terminal de MacOS o Linux:**
`python3 nombre_de_archivo.py`

*Ejemplo:* 
*python3 quiz_git.py*
